## jSprite

Quick and easy JavaScript sprites for use in your html.

````
let mySprite = new jSprite({
    spriteSheet: "./images/sheet1.png",
    container: "sp1",
    columns: 12,
    rows: 7,
    widthOffset: -6,
    timing: 30,
    // timings: [100,200,300,400,500,600],
    repeat: true
});

````
